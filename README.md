# Claude Code Skills

Personal workspace built with Claude Code — contains custom skills for automating tasks.

## Skills

### 1. Weather API (`skills/weather-api/`)
Fetches real-time weather data from OpenWeatherMap API for any city.
- **Platform**: OpenWeatherMap REST API
- **Run**: `python weather.py`

### 2. CSV Analyzer (`skills/csv-analyzer/`)
Reads a CSV file, computes statistics, and exports a Markdown report.
- **Run**: `python analyze.py data/sample_sales.csv`
